O sistema consiste em
pegar o arquivo usuario.txt, tratar os dados dentro dele,
pegar o arquivo arm_usuarios.txt, tratar os dados dentro dele,
ultilizando os dados de armazenamento dentro de arm_usuarios.txt para criar uma coluna
de relação ao usu do armazenamento de cada usuario e o armazenamento total do servidor
representado por porcentagem,
pegar os dados das 3 tabelas e através dos ids especificado em usuarios.txt da coluna IDs e
os ids em arm_usuarios.txt da coluna Id_usuaruio e atraves dessa relação mesclar as 3 tabelas
a tabela de porcentagem que é criada herda os id_usuario da tabala arm_usuarios.txt
depois de mesclar todas as tabelas e preciso pegar os dados e representalos no arquivo relatorio.txt
O sistema tambem realiza a conversão de bytes que é o tipovalor de armazenamento de cada
usuario que esta  na tabela arm_usuarios.txt e convert para Gigabytes, Megabytes ou Kilobytes
